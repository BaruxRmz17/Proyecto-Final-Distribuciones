# Proyecto Final de Probabilidad: Distribuciones Exponencial y Uniforme

## Objetivo General del Proyecto

Este proyecto se enfoca en el **análisis teórico, la aplicación práctica y la simulación computacional** de dos distribuciones fundamentales en la estadística y la ingeniería: la **Distribución Exponencial** y la **Distribución Uniforme Continua**.

El objetivo es conectar los conceptos matemáticos con su aplicación en fenómenos reales (tiempos de espera, confiabilidad, generación de números aleatorios) utilizando Python para la simulación y validación de resultados.

---

##  Estructura del Proyecto

El proyecto está dividido en cuatro partes principales, de acuerdo con el plan de estudios, documentadas en el PDF final y simuladas en el archivo principal de Python (`simulacion_distribuciones.ipynb` o similar).

| Parte | Título | Contenido Principal | Entregable Clave |
| :--- | :--- | :--- | :--- |
| **Parte 1** | **Investigación Teórica** | Descripción formal (PDF, CDF), parámetros (tasa $\lambda$, $a, b$), momentos (media, varianza) y propiedades clave (falta de memoria). | 10 ejercicios resueltos. |
| **Parte 2** | **Aplicaciones Reales** | Investigación de 3 aplicaciones reales por distribución (e.g., Teoría de Colas, Confiabilidad, Generación de Aleatorios). | Justificación y ejemplos numéricos. |
| **Parte 3** | **Simulación en Python** | Generación de muestras aleatorias, visualización gráfica y comparación de estadísticos muestrales vs. teóricos. | **Código Python Comentado y Funcional.** |
| **Parte 4** | **Conclusiones** | Análisis comparativo de ambas distribuciones y reflexión sobre la convergencia de la simulación. | Análisis crítico final. |

---

##  Tecnologías y Dependencias

El código de simulación fue desarrollado para garantizar la portabilidad y eficiencia, utilizando el *stack* de ciencia de datos estándar en Python.

* **Lenguaje:** Python 3.x
* **Simulación/Cálculo:** `NumPy`
* **Estadísticas Teóricas:** `SciPy.stats` (para funciones de densidad y CDF)
* **Visualización:** `Matplotlib` y `Seaborn` (para gráficos de alta calidad)
